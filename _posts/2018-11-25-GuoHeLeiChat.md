上周郭贺磊从华山来北京政审别人，顺道找我。  在融科前边Google的标志处碰头，迎面就是一句非常炸裂的话：

``` yaml
你的问题我认为都不是问题，你解决一下我的问题。
```

这是什么瞎操作！！！  

问题就是：
<ol>
    <li><b>现在转技术类工作；专业书看不懂，也记不住。</b></li>
    <li><b>跳到一个陌生的领域，很心慌，搞不定会不会完蛋。</b></li>
</ol>

虽然他放弃了我，但是我不能放弃他。  毕竟是“从你去年离开华山以后，我就一直在积累生活”的男人。  

姑且把这两个问题重新整理成以下三个问题：  

<ol>
    <li><b>怎么读书/学习</b></li>
    <li><b>为什么读书/学习</b></li>
    <li><b>面对未知的恐惧</b></li>
</ol>

全是哈姆雷特式的问题，很大。  
任何一种具体的，个人化的答案，意义可能就是抛砖引玉，brain storm吧。  

分享一些个人感受，千金买骨。  

# 一
<b>怎么读书</b>这件事，  
上周老大在损我【他是真的爱损我我也木有办法】的时候,  
捎带了两句干货，

<ol>
  <li><b>先见树林，后见树；再被吊死了，你也没什么遗憾的</b>;</li>
  <li><b>循环迭代，知行合一，不断扬弃</b>;</li>
</ol>

最早开始做调研，看到师兄在做链路预测，就想往上靠。  
清华当时牛人，主要做network embedding（NE）的。  
我也就看了两篇他的文章，然后就着急忙慌地想做NE， 
结果就被老大毫不留情地嘲讽了：  

``` yaml
你不要上来就告诉我你要做什么点，
你连这个研究领域的整个树林都没看见，
就着急忙慌地找棵树把自己吊死
```

树林？？！！还有这种视角！

然后我就跳进了知乎，  
国外知乎quora，  
medium，  
google scholar中.  
扒拉这个事情，有  
哪些基本概念，  
哪些大佬，  
哪些机构，  
哪些会议，  
哪些期刊，  
哪些应用，  
每一个大佬都讲过什么，  
这些期刊都讲的什么话题。。。

最终得出一个结论：不跳NE这个坑！

上学以来形成的习惯，  
就是找一本教科书，  
然后开始啃开始撸，  
终于啃不动了，  
就怀疑自己是不是笨【及其有可能的我！】，  
是不是不适合，  
是不是基础不行。  

姑且不说他有可能就是写的不好。  
一个我跳过的坑是，  
讲的东西太老太底层你基本不会直接用到，  
完全没有必要上来就搞透每一个定理，  
可能更重要的事情，  
是这个领域的知识体系是什么，  
有哪些基本概念和工具，  
哪些机构和大佬在做，  
他们说过什么，有什么产品，未来什么规划，  
我怎么和他们勾搭上。。。 :sunglasses:  

看书看论文，经常会质疑人生：  
没什么用啊不懂啊。  
然后,  
又被老大嘲讽了 :sweat:  

``` yaml
不看论文不行，光看论文非看傻啊。
先七手八脚做一个弱智的东西出来。
然后在上边改，努力提升性能。
一边改一边看，循环迭代几次，
你的能力就上去了。
其实看文章，最后一定是看完了，
实验步骤结果直接就在脑子里有了，
只不过跑一下验证自己的东西而已。
```

关键是做啊！！！  

刚开始很痛苦，不知道从哪下手。  
我之前没有接触过一点数据库的东西，
突然跳进PostgreSQL这个坑，  
简直就是大型车祸现场...  
但是忍一忍，
终究还是能从坑里爬出来的，  
然后。。。  
就还挺好玩的！  
再然后，
看工业界和学术界，对同一个问题的不同操作，  
会发现一些不可告人的秘密。。。  

毕竟，  
读博也好读书也好，  
乐趣不就是自己找个坑跳进去，  
然后看看能不能活下来看到明天的太阳，  
来一杯马蒂尼么？  
真的刺激！  

## 二
<b>为什么读书</b>这件事，  
其实不光是为了记住。  

因为，  
没有感受过，体验过，  
怎么谈理解，  
没有理解，  
怎么谈记住。  

说点听起来很欠揍的东西吧。

<b>孤独</b>！

周国平谈过，他讲，  
灵魂只能独行。  
无论多么亲密的两个人，  
只不过是黑夜中相伴着朝圣，  
朝自己灵魂的圣，  
别人没的理解。  

王小波谈过，  
不过，  
我就记住他的土味情话了：  
你要是喜欢别人我会哭，但是还是喜欢你。  

还有很多人，  
都谈过，  
也都谈的很好。  

但只有叶嘉莹先生的话，  
我有共鸣：
``` yaml
我已经九十多岁了，回顾我平生走过的路，是中国的古典诗词伴随了我的一生。
```
``` yaml
于我而言，诗词的研读并不是我追求的目标，而是支持我走过忧患的一种力量。
```

八月仲夏夜，  
在满都海公园跑步，  
在草原上飙车到180kmh，  
去姥姥家蹭各种好吃的东西，  
跟着妈妈去各种奇奇怪怪的地方，  
这些让我很开心。  
慢慢地，
开心也会让人生出少年维特式的烦恼。  
这一切生活在我生活里的美好，  
陪伴着我成长的周遭，  
都不会陪我走完全程。  
虽然不舍，  
可是终有告别时刻。

有什么东西，  
是我生不带来，死会带走的？  
有什么东西，  
是我被所有人喜欢也会讨厌也好，  
或者，没有讨厌没有喜欢，只是单纯地不care也好，  
都会坚定左右的？  

``` yaml
我已经九十多岁了，回顾我平生走过的路，是中国的古典诗词伴随了我的一生。
```
``` yaml
于我而言，诗词的研读并不是我追求的目标，而是支持我走过忧患的一种力量。
```

可能，也就是你看过，想过的书了。  

一个常解，  
就是学习、读书是一件特别高尚厉害的事，  
毕竟是要“为天地立心，为生民立命，为往圣继绝学，为万世开太平”的。  

可能这对一部分人是对的，  
比如，  
卡文迪许，  
冯诺伊曼，  
牛顿，  
比如谢灵运，  
比如曹子建，  
比如既是权贵又有无限的闲激情又有天赋的曹子建们。。。  

这部分人里肯定没我。

是权贵，你就不愁生计，随心所欲；
有闲置的激情，你才会被一些没什么用的东西吸引；
有天赋，你才会把这些没什么用的东西，做的让后人想起来都是金光闪闪自带BGM似的。

那我是一个都不具备的普通白丁啊，
白丁读书，
要什么有用，
要的是有趣，
要的是一个一生陪伴啊。

有些说小不小的小事，  
可以过的很大，  
披浪漫主义的披风，  
骑个人英雄主义的马。  

你不需要找个王健林似的爹，实现王健林似的小目标才算个人成功；  
你在你的抉择里你可以选择秦桧式的逃避，或者岳飞式的勇敢；  
你在你的困厄里可以选择文天祥式的“读圣贤书，所谓何事，庶几往后几无悔矣”；  
你在你的格格不入里可以选择陶潜式的完成，陶潜式的斑斓；  
你也可以在你的夜想里，选择西门庆式的。。。好像过分了。。。。。。  

``` yaml
真正伟大的诗人，都是在用自己的生命来写作自己的诗篇的，
都是在用自己的生活来实践自己的诗篇的
```

你可以在他们的书里找到你，你也可以在你自己的生活里找到他们。

# 三
面对未知的恐惧。  
原文引用直10武装直升机总师吴希明的两段话：  

``` yaml
如果你害怕，你要仔细想一想为什么害怕，
是很多东西你没做到，或者你不愿意去做，
然后又是侥幸心理，
然后期望着奇迹发生，
然后害怕奇迹不发生，
才会这样。
但是如果你所有你的认真努力，
未知道的事情，
你努力去了解，
努力去掌控，
甚至哪怕差一点，
把所有的事情能想到的尽量想到，
做到极致，
我没办法再做了，
虽然很多的不懂，
但是我应该做的我都做了。
那就这样了，
你也不会害怕。
```

``` yaml
人所有的恐惧，
都是对未知的东西才会恐惧，
他为什么害怕，
怕的是还有什么事自己没做到，
还有什么东西没想到，
什么东西心中不可预测，
人就会害怕，
但是如果说你做的一件事情，
你把所有能想到的东西，
大家都想到以后，
然后想的东西，
我们都全力以赴去证明以后，
他不应该是害怕。
```

没什么好解释的，去做就是了。  

杨绛先生回复青年学生：你的问题，是想的太多而读书太少。  
我们的问题，是想的太多，做的太少。  

给自己定个底线，就去做，  
坚持过去一次，  
只要一次，  
看问题的眼神都变了。
